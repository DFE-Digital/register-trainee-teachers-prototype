How to add trainee information to the bulk add new trainee CSV template

Training providers can register their trainees in bulk by creating trainee records and adding the required data.

Important

This draft guidance is for review and testing purposes only. Send your feedback or questions to Registertraineeteachers@education.gov.uk.

CSV bulk add new trainees will be available in Register trainee teachers (Register) before the start of the 2025 to 2026 academic year.

Download an empty CSV file template

You must only use this CSV file template to register new trainees in Register trainee teachers (Register).

If you use this to update already existing trainees, it will come back as an error.

The CSV template uses HESA reference data.

Download empty bulk add new trainees CSV template

Use this guidance to complete the CSV file template

Use the guidance to check how new trainee data maps between your student record system and Register, using the CSV upload found in the ‘Bulk updates section’ in Register.

If mandatory fields are not filled in correctly, the CSV upload will come back as an error.
